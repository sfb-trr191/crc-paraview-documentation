Paraview for Windows

This is a programm that automatically installs a desired version on paraview on your Windows computer. The programm also automatically checks for updates on new plugins provided by us for the Windows version of paraview.

Requirments

To run the programm successfully you need to install a few additional programms

  • GIT
  • Python

Installation

Download the git repository Paraview-Windows and place it somewhere on your computer. Execute the paraview.bat script inside the project and follow the instructions on the command shell. The download and installation may take a while depending on your internet connection. Now create a link to the paraview.bat script, you should be able to run paraview.

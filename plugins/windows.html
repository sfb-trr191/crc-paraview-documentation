Paraview Plugins for Windows

We provided a number of plugins for Windows on our Gitlab Repository that will automatically download and update whenever you run paraview.

Custom Plugins

You can also intigrate your own plugins using the programm. For this you either need the binary files .so that correlate with your paraview version or intigrate python plugins .py. For the binary plugins: create a folder in the Mac plugin folder which has the same name as the plugin

If your plugins require any additional libraries you can copy those into the lib folder the same way

For custom python plugins exactly the same but in the folder "Python"

If you want to add your own VTK plugin, you need to copy the .xml file into a directory with the same name
